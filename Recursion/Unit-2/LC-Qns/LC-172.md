##### Normal Recursion Solution

```java
class Solution {
    public int trailingZeroes(int n) {
        return helper(n, 5);
    }
    int helper(int n, int v){
        if(v > n) return 0;
        return n/v + helper(n,v*5);
    }
}
```

##### Tail Call Recursion Solution

```java
class Solution {
    public int trailingZeroes(int n) {
        return helper(n, 5, 0);
    }
    int helper(int n, int v, int a){
        if(v > n) return a;
        a = a + n/v;
        return helper(n,v*5,a);
    }
}
```

##### CP Solution

![[Pasted image 20260104184656.png]]
```java
class Solution {
    public int trailingZeroes(int n) {
        return n/5 + n/25 + n/125 + n/625 + n/3125;
    }
}
```
